# Заметки к выпуску v0.2.0

### ✅ Реализовано:
1. **Интерфейс:** Современный дизайн на Tailwind v4.0. Добавлена таблица с чередованием цвета строк ("зебра").
2. **Пагинация:** Выбор количества записей (10, 20, 50) и кнопки переключения страниц.
3. **Фильтрация:** Реализованы 4 режима (Все, Доступен, Ремонт, Продан). Логика фильтрации вынесена на сторону сервера.
4. **Валидация:** Форма блокирует отправку некорректных данных (короткий VIN, старый год выпуска).
5. **Данные:** Скрипт генерации (`seed.ts`) теперь создает 50 случайных объектов для проверки нагрузки и поиска.

### ⚠️ Зафиксированные баги (As Designed):
- Расхождение названий статусов в фильтре и таблице ("Ремонт" vs "В ремонте"). Оставлено для отработки неточного поиска в автотестах.

### 📅 План на следующий этап:
- Реализация системы входа (Login) через JWT в защищенных куках.
- Разделение прав доступа (Админ / Механик / Гость).
